# Entry 4 : FP Tool Bulma
##### 3/14/26

# Content

In this blog entry I started learning about my FP tool, which is Bulma, a CSS framework used to design and organize websites. Furthermore, Bulma is a CSS framework that uses flexbox to help developers create a responsive layouts for their web interface. Additionally, to start learning Bulma, I read the installation guide provided by Bulma documentation to understand how to add Bulma to a project. Moreover, Bulma allows the framework to style the webpage without knowing a single thing about CSS and even writing a large amount of CSS. I also discovered learning resources connected to CSS and frontend development from the Bulma website, which helped me understand how frameworks are used in projects. While researching Bulma, I noticed that the layout system is somewhat similar to Bootstrap, which we previously used in class. For example, Bulma uses columns and containers to help structure a webpage so that it can adapt and work well on different types of screen sizes, whether it can be a phone, ipad, computer, etc.

# Engineering Design Process

Currently I am still researching about my FP tool and I am learning how my FP tool works, and how it can help me build my Freedom Project. I explored Bulma examples to understand how its layout and how frontend components function.
The next stage I need to take is going to be the planning stage, where I will start deciding how Bulma will be used in my Freedom Project. How I will plan to organize my layout using columns,rows, and other Bulma components.
